# CSV 배치 안내

현재 사용자가 보유한 CSV는 아래 2종입니다.

1. `경기도_김포시_교통약자이동지원센터정보_20260105.csv`
2. `경기도_김포시_노인장애인보호구역_20251222.csv`

사용자가 같은 보호구역 CSV명을 두 번 언급했지만, 실제 파일은 한 개만 두면 됩니다.
중복 파일은 만들 필요 없습니다.

## 배치 경로

```text
/data/경기도_김포시_교통약자이동지원센터정보_20260105.csv
/data/경기도_김포시_노인장애인보호구역_20251222.csv
```

## 활용 방식

- 교통약자 이동지원센터 CSV: 이동지원센터명, 전화번호, 운영시간, 차량 현황 fallback
- 노인장애인보호구역 CSV: 보호구역 인접 위험도, 보행 안전 보정, B2G 대시보드 지표

CSV 파일이 없거나 파싱 실패하면 mock fallback으로 전환해야 합니다.

## 반다비 프로젝트에서 가져갈 부분

### 1. 입력 추출 패턴

```python
def extract_question(value):
    if isinstance(value, dict):
        return value.get("input") or value.get("question") or value.get("query") or ""
    return str(value)
```

### 2. 검색 근거 포맷팅 패턴

```python
from pathlib import Path

def format_docs(docs):
    formatted_docs = []
    for idx, doc in enumerate(docs, start=1):
        source = Path(doc.metadata.get("source", "unknown")).name
        page = doc.metadata.get("page", None)
        page_text = f"{page + 1}쪽" if isinstance(page, int) else "위치 정보 없음"
        content = doc.page_content.strip()
        formatted_docs.append(f"[근거 {idx} | {source} | {page_text}]\n{content}")
    return "\n\n".join(formatted_docs)
```

### 3. 문서 분할 개념

원본은 chunk_size=900, chunk_overlap=150, separators 기반 분할을 사용했습니다.
반다비 프로젝트에서는 Markdown heading + 문단 기준 chunking을 기본으로 하고, 필요 시 아래 개념만 참고합니다.

```python
chunk_size = 900
chunk_overlap = 150
separators = ["\n\n", "\n", ". ", "? ", "! ", " ", ""]
```

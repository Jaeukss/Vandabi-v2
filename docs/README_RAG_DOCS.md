## 사용 위치

Streamlit 프로젝트에서는 아래 경로에 넣는 것을 권장한다.

```text
C:\Users\User\Desktop\반다비 스트림릿\docs
```

## RAG 구현 권장

- 기본 검색: BM25
- LLM 답변 생성: OpenRouter 무료 모델 가능
- 실패 시 fallback: 검색 문단 기반 템플릿 답변

## 답변 원칙

- 문서에 있는 내용만 근거로 답한다.
- 이동지원 실행, 이용 가능 여부 결정, 건강 관련 최종 결정, 법적 판단은 단정하지 않는다.
- 문서에 없는 내용은 “현재 등록된 문서에서 확인되지 않습니다.”라고 답한다.
- 변경 가능성이 있는 운영시간, 요금, 접수 일정은 최종 확인 필요 문구를 붙인다.
